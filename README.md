# DEVO-AI-OS v1.0

AI Creative Operating System for Story, Visual, Video, Music, Education, Canva and GitHub workflows.

## Core Formula

```text
Complex Knowledge
→ Story
→ Image
→ Experience
→ Change
```

## API Endpoints

```text
/api/openai
/api/router
/api/engines
/api/workflows
/api/quality
/api/canva
/api/github
/api/health
```

## Local Run

```bash
npm install
npm run dev
```

## Vercel Setup

Add Environment Variables:

```text
GITHUB_TOKEN=your_fine_grained_github_token
GITHUB_OWNER=rameau65
GITHUB_REPO=DEVO-AI-OS
GITHUB_DEFAULT_BRANCH=main
GITHUB_API_SECRET=your_secret
```

## GitHub Refactor → Commit → PR

```bash
curl -X POST https://devo-ai-os.vercel.app/api/github \
-H "Content-Type: application/json" \
-H "x-devo-secret: YOUR_SECRET" \
-d '{
  "action":"refactor_commit_pr",
  "taskName":"phase1-test",
  "newBranch":"feature/phase1-test",
  "files":[
    {
      "path":"docs/phase1-test.md",
      "content":"# Phase 1 Test\n\nGenerated by DEVO-AI-OS.",
      "message":"Add phase 1 test document"
    }
  ],
  "prTitle":"Add Phase 1 GitHub API test",
  "prBody":"Automated PR generated through DEVO-AI-OS GitHub API."
}'
```
