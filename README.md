# DEVO-AI-OS v1.1

AI Creative Operating System with:

- GitHub automatic commit
- GitHub PR creation
- Vercel automatic deploy hook
- Canva Design Agent
- OpenAI API
- MCP-compatible endpoint
- Workflow Router
- Agent Memory

## Deploy

Import this repository into Vercel.

## Environment Variables

```text
DEVO_API_SECRET=change_this_secret
GITHUB_API_SECRET=change_this_secret
MCP_API_SECRET=change_this_secret

GITHUB_TOKEN=your_fine_grained_github_token
GITHUB_OWNER=rameau65
GITHUB_REPO=DEVO-AI-OS
GITHUB_DEFAULT_BRANCH=main

OPENAI_API_KEY=your_openai_api_key
OPENAI_MODEL=gpt-4.1-mini

VERCEL_DEPLOY_HOOK_URL=https://api.vercel.com/v1/integrations/deploy/...
VERCEL_PROJECT_URL=https://devo-ai-os.vercel.app
```

## GitHub Auto Commit + PR

```bash
curl -X POST https://devo-ai-os.vercel.app/api/github \
-H "Content-Type: application/json" \
-H "x-devo-secret: YOUR_GITHUB_API_SECRET" \
-d '{
  "action":"refactor_commit_pr",
  "taskName":"test-auto-pr",
  "newBranch":"feature/test-auto-pr",
  "files":[
    {
      "path":"docs/test-auto-pr.md",
      "content":"# Test Auto PR\n\nGenerated by DEVO-AI-OS v1.1.",
      "message":"Add test auto PR document"
    }
  ],
  "prTitle":"Test DEVO-AI-OS Auto PR",
  "prBody":"Automated PR generated by DEVO-AI-OS v1.1.",
  "triggerDeploy": false
}'
```

## Vercel Deploy Hook

```bash
curl -X POST https://devo-ai-os.vercel.app/api/vercel \
-H "Content-Type: application/json" \
-H "x-devo-secret: YOUR_DEVO_API_SECRET" \
-d '{"action":"trigger_deploy"}'
```
