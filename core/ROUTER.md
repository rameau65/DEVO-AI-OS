# Engine Router

Route user requests automatically.

- Story → STORY_ENGINE
- Visual/Image → VISUAL_ENGINE
- Video → VIDEO_ENGINE / SEEDANCE_ENGINE
- Music → MUSIC_ENGINE
- Meditation → MEDITATION_ENGINE
- Infographic → INFOGRAPHIC_ENGINE + CANVA_AGENT_ENGINE
- Presentation → CANVA_AGENT_ENGINE
- GitHub → GITHUB_ENGINE
- Final review → QUALITY_ENGINE
